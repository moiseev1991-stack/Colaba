# A/B прогон промпта КП — как запустить

Скрипт `scripts/kp_ab_compare.py` генерирует для 5 отобранных стоматологий 2 варианта КП:
- **A** — старый свободный промпт (текущий продакшн)
- **B** — новый каркас «4 хода», канал мессенджер, микрошаг «созвон 10 минут»

Кладёт результат в Markdown, чтобы сравнить глазами.

## Стоимость

- 5 компаний × 2 варианта = **10 LLM-запросов** через ProxyAPI.
- ~5-7 руб за запрос → **50-70 рублей** за прогон.

## Запуск на проде

```bash
ssh spinlid
cd /opt/colaba
docker compose -f docker-compose.prod.yml exec -T backend \
  python scripts/kp_ab_compare.py > /tmp/kp_ab_report.md
scp root@88.210.53.183:/tmp/kp_ab_report.md ./docs/
```

Файл `docs/kp_ab_report.md` уходит на просмотр — в нём каждая компания
с subject/body для варианта A и B рядом. Смотришь и решаешь: новый
промпт стал лучше? Если да — 4hods можно включать по умолчанию для новых
КП. Если нет — крутим справочники / промпт-tail дальше.
