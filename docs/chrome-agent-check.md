# Chrome-агент: чек-лист проверки spinlid.ru

**Версия:** 2026-06-05 (после релиза 1.48.2+ — bulk-CSV / чистка пресетов / Yandex city).

## Контекст для агента

Ты — Chrome-агент, который должен проверить работу production-сайта **https://spinlid.ru** (lead-gen сервис «Colaba» для B2B-продаж в России). Залогиниться сейчас нечем — **demo-аккаунт ещё не сделан**. Поэтому проверяй только то, что доступно без авторизации, плюс убедись что вход работает технически (форма принимает данные, нет 500-х).

В конце пришли отчёт ровно в формате:

```
✅ Работает: <пункт> — <короткая заметка что увидел>
⚠️  Странность: <пункт> — <что не так, скриншот в name.png>
❌ Сломано: <пункт> — <как воспроизвести, скриншот>
🤷 Не смог проверить: <пункт> — <почему>
```

Скриншоты прикладывай только к ⚠️ и ❌. Не описывай каждый успешный клик — только «работает» + одна строка.

---

## 1. Landing-страница (без логина)

Открой **https://spinlid.ru** в чистой инкогнито-вкладке.

- [ ] **1.1** Страница загружается без ошибок в консоли (открой DevTools → Console; красные ошибки = ⚠️).
- [ ] **1.2** Hero-блок виден: заголовок, подзаголовок, кнопка призыва к действию.
- [ ] **1.3** Прокрути вниз — есть секции «Как работает», «Модули», «Тарифы».
- [ ] **1.4** Скролл плавный, нет «прыжков» layout-shift.
- [ ] **1.5** Все картинки и иконки загрузились (нет «битых» рамок).
- [ ] **1.6** Тёмная тема (если есть переключатель) — переключается без поломки вёрстки.
- [ ] **1.7** На мобильной ширине (DevTools → Toggle device toolbar → iPhone 12) меню/блоки адаптируются.

## 2. Регистрация и вход

## 3. Поиск компаний (модуль Maps) — главная фича

> Это сердце продукта. Проверь подробнее всего.

### 3.1 Новый поиск

- [ ] **3.1.1** Открой раздел «Поиск компаний» (или «Карты», в меню слева).
- [ ] **3.1.2** Видны поля: «Ниша», «Город», «Источники (2GIS / Я.Карты)».
- [ ] **3.1.3** Введи **«стоматология»**, город **«Балашиха»**, выбери оба источника. Нажми «Найти».
- [ ] **3.1.4** Появляется лента поиска. Карточки приезжают постепенно (не блок).
- [ ] **3.1.5** Над лентой видишь сегмент-переключатель **«Все · 2GIS · Я.Карты»** с цифрами.
- [ ] **3.1.6** Дождись завершения (статус «Готово»).

### 3.2 Фильтры и пресеты

В левой панели:

- [ ] **3.2.1** Видны 6 готовых пресетов:
  - Кризис репутации
  - Падение рейтинга
  - Нужен сайт
  - Нужна автоматизация
  - Узкий ассортимент
  - Богатый ассортимент
  
  **Если видишь старые пресеты «Стабильный», «Хаос в работе», «Готовы к автоматизации», «Есть сайт» — это ⚠️**, фронт не пересобрался.

- [ ] **3.2.2** Кликни «Нужен сайт» — фильтрация применилась, в выдаче только компании без сайта (бейдж «нет сайта»).
- [ ] **3.2.3** Кликни ещё раз — фильтр снят.
- [ ] **3.2.4** Открой блок «ПЛАТЁЖЕСПОСОБНЫЕ (DADATA)» (если виден) — поля «Оборот от» и «Возраст».

### 3.3 Карточка компании

- [ ] **3.3.1** Кликни любую карточку — справа выезжает Drawer с подробной информацией.
- [ ] **3.3.2** В Drawer: контакты (телефон, сайт, email, мессенджеры), отзывы, ссылки на 2GIS/Я.Карты.
- [ ] **3.3.3** Email-ы НЕ содержат `help@2gis.ru`, `info@2gis.com`, `support@2gis.ru`, `noreply@2gis.ru` (после релиза 1.48.1+ они отфильтрованы).
- [ ] **3.3.4** Закрой Drawer кликом по крестику или клавишей Esc.

### 3.4 Multi-source: фильтр источника

- [ ] **3.4.1** В шапке кликни **«2GIS»** — остаются только 2GIS-карточки. URL меняется на `?src=2gis`.
- [ ] **3.4.2** Кликни **«Я.Карты»** — URL = `?src=yandex_maps`.
- [ ] **3.4.3** Браузерная кнопка «Назад» возвращает к предыдущему фильтру источника.
- [ ] **3.4.4** Для **«стоматология / Балашиха / Я.Карты»** в выдаче **НЕ ДОЛЖНО быть карточек из Химок / Мытищ / Реутова** (после релиза 1.48.2). Адреса всех видимых компаний содержат «Балашиха».

### 3.5 Bulk-операции (чекбоксы)

- [ ] **3.5.1** У каждой карточки в левом верхнем углу есть чекбокс.
- [ ] **3.5.2** Отметь 3-5 карточек. Над лентой появляется панель «Выбрано: N».
- [ ] **3.5.3** В панели есть кнопки **«Добавить выбранные в список»** и **«Очистить»**.
- [ ] **3.5.4** Кнопка «Очистить» снимает все отметки.
- [ ] **3.5.5** Кнопка «Выбрать все на странице» отмечает все видимые карточки одним кликом.

### 3.6 Экспорт

В шапке выдачи должна быть **одна** кнопка **«📥 Экспорт ▾»** (НЕ две отдельные «Экспорт CSV» + «Excel: лиды на сайт»).

- [ ] **3.6.1** Кликни «📥 Экспорт ▾» — открывается дропдаун с тремя пунктами:
  - **CSV — все компании** (с учётом фильтров)
  - **CSV — выбранные (N)** (disabled пока нет отмеченных)
  - **💼 Excel — лиды на сайт**
- [ ] **3.6.2** Кликни «CSV — все компании». **⚠️ Chrome-агент не скачивает файлы** — просто пометь как «работает, диалог скачивания появился» (или 🤷 если не появился).
- [ ] **3.6.3** Отметь 2 карточки → снова открой «📥 Экспорт ▾» → «CSV — выбранные (2)» уже не disabled.
- [ ] **3.6.4** Клик по «💼 Excel — лиды на сайт» — диалог скачивания появляется.

### 3.7 Карта

- [ ] **3.7.1** Кликни «Карта» в шапке — лента сменяется на карту с пинами.
- [ ] **3.7.2** Карта рендерится без серых пустых тайлов.
- [ ] **3.7.3** Клик на пин открывает popup с названием компании.
