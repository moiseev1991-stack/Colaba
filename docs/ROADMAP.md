# Colaba — Дорожная карта

**Последнее обновление:** 9 июня 2026
**Текущая версия:** v1.68.0

---

Проект прошёл большой путь от горизонтального «поиск + SEO + email» до глубокого **Maps-пайплайна с AI-диагностикой болей**. Сейчас ядро продукта:

> **2GIS + Яндекс.Карты → отзывы → AI-анализ болей → ЛПР → outreach**

## Среднесрочные цели (июль 2026)

### После замыкания пайплайна

1. **Multi-city × Multi-niche dashboard** — отчёт по массовым прогонам
2. **Auto-обогащение (enrichment)** — Контур.Фокус / List-org для ИНН/контактов
3. **AI-генерация КП (proposals)** — из собранных данных генерировать коммерческие предложения
4. **CRM-light** — жизненный цикл лида: холодный → тёплый → встреча → сделка

### Маркетинг

5. **Telegram-канал «прогонов»** — публичные кейсы использования
6. **Партнёрская программа 30%** — реферальная система
7. **Посадочные под ниши** — отдельные лендинги для стоматологий / автосервисов / фитнеса

## Что НЕ берём (сознательно)

| Технология / модуль | Причина |
|---------------------|---------|
| WhatsApp-рассылки | Meta заблокирован в РФ, B2B не канал |

| OAuth VK / Telegram | B2B так не логинятся |
| Reputation monitoring | Отдельная ниша, не сейчас |
