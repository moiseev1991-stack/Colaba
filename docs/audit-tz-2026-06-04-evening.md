# Аудит spinlid.ru → «По картам» (2026-06-04 вечер)

Скопируй блок ниже целиком в Claude in Chrome (расширение/приложение,
которое умеет открывать сайт и кликать). Это самостоятельный prompt —
ничего больше присылать не нужно.

---

## Контекст

Colaba — B2B leadgen-инструмент. Раздел «По картам» парсит компании из
**2GIS** и **Я.Карт**, тащит контакты с обоих источников, ставит
AI-диагноз по болям из отзывов.

С прошлого аудита (2026-06-02) выкатили **пакет изменений**:

1. **Multi-source фильтр.** Сегмент `Все · 2GIS · Я.Карты` в шапке
   выдачи и дублирующий дропдаун «Источник» в боковой панели фильтров.
   При клике URL получает параметр `?src=2gis` / `?src=yandex_maps`.
2. **Фокус карточки на выбранном источнике.** У склеенной компании
   (есть в обоих источниках) при фильтре «Только 2GIS» рейтинг, отзывы
   и контакты подменяются на 2GIS-цифры; бейдж «2GIS» подсвечивается,
   «Я.Карты» приглушается. При «Только Я.Карты» — наоборот.
3. **Тепловая карта удалена.** Кнопки «🔥 Тепло» в шапке больше нет
   (была между «Карта» и «Экспорт CSV»). Это намеренно.
4. **Кнопка «💼 Excel: лиды на сайт» удалена** из шапки.
5. **CSV-экспорт с UTF-8 BOM и `;`-разделителем** — теперь Excel в
   русской локали открывает CSV нормально (раньше были кракозябры
   «РўРѕ»).
6. **Yandex-отзывы починены.** Yandex закрыл публичный AJAX-endpoint;
   парсер переписан на SSR-HTML (`yandex.ru/maps/org/{id}/reviews/`).
   У 153 yandex-профилей теперь подтянуто 7393 отзыва. В drawer-карточке
   во вкладке «Я.Карты» должны быть реальные отзывы — раньше там
   везде было «0 отзывов».

ЛОГИН: <подставь сам>
ПАРОЛЬ: <подставь сам>

URL: https://spinlid.ru/app/leads

После авторизации **обязательно сделай Ctrl+Shift+R** на странице
`/app/leads`, иначе браузер покажет старый закешированный bundle.

## A. Двух-источниковый поиск

1. Жми **«Новый поиск»**.
2. Ниша: `стоматология`. Город: `Балашиха`.
3. На форме поиска должны быть выбраны **оба источника** (2GIS + Я.Карты).
4. Жми «Запустить». Дождись завершения (1-3 минуты для Балашихи,
   из кэша — секунды).
5. Статус-строка в шапке: `Источники: 2gis,yandex_maps. Статус: …
   Найдено компаний: N`.

---

## B. Шапка выдачи — что должно быть, чего быть не должно

В шапке выдачи смотри на ряд кнопок справа от заголовка
«стоматология — Балашиха».

**Должно быть** (5 кнопок + сегмент-переключатель):
- Сегмент-переключатель `Все N · 2GIS X · Я.Карты Y` (виден только
  когда оба источника не пустые).
- Кнопка `📋 Список`.
- Кнопка `🗺️ Карта`.
- Кнопка `Экспорт CSV`.
- Кнопка `Новый поиск`.

**НЕ должно быть:**
- Кнопки «🔥 Тепло» — удалена.
- Кнопки «💼 Excel: лиды на сайт» — удалена.

🔴 **БАГ если**: видишь хоть одну из удалённых кнопок. Это значит браузер
показывает закешированный bundle — сделай Ctrl+Shift+R ещё раз. Если
после перезагрузки они всё ещё есть — это реальный баг, репорт.

### B1. Tooltip сегмент-переключателя

Наведи курсор на сегмент `Все N · 2GIS X · Я.Карты Y`. В tooltip
должна быть строка:

> Найдено: всего N · 2GIS X · Я.Карты Y · в обоих Z

Где Z — это компании, найденные в обоих источниках (склеенные).
Z ≥ 0 — норма.

## C. URL-параметр `?src=` (URL-persistence)

1. Сейчас URL: `…/app/leads` (без `?src=` или с `?src=all`).
2. Кликни **«2GIS»** в сегмент-переключателе.
   - Список сокращается (исчезают компании только-из-Я.Карт).
   - **В адресной строке появляется `?src=2gis`**. Пример:
     `https://spinlid.ru/app/leads?src=2gis`.
3. Кликни **«Я.Карты»** → URL: `?src=yandex_maps`.
4. Кликни **«Все»** → `?src=` исчезает (или становится `?src=all`).

### C1. Назад/Вперёд браузера

1. На «Все» → клик «2GIS» (URL: `?src=2gis`).
2. На «2GIS» → клик «Я.Карты» (URL: `?src=yandex_maps`).
3. Нажми кнопку «Назад» в браузере → URL должен вернуться на
   `?src=2gis`, сегмент-переключатель подсветить «2GIS», список —
   переключиться обратно.
4. «Вперёд» → `?src=yandex_maps` + Я.Карты.

### C2. Дублирующий фильтр в боковой панели

В левой панели фильтров (где «Минимум отзывов», «Ответы владельца»,
«Сайт») должен быть дропдаун **«Источник»** с тремя вариантами:
`Все источники / Только 2GIS / Только Я.Карты`.

- Переключение в дропдауне синхронно с сегментом в шапке: клик на
  «Только 2GIS» в дропдауне → подсветка «2GIS» в шапке и наоборот.

---

## D. Фокус карточки на выбранном источнике

Найди в выдаче **склеенную компанию** — у неё в правом нижнем углу
карточки **два бейджа подряд**: `2GIS · Я.Карты`. Эталоны для
Балашихи: `Стоматологическая Практика 14*14`, `Dental Harmony`,
`Дентека`. У них в шапке справа стоят два бейджа.

### D1. На «Все» (нейтрально)

1. Запиши: рейтинг, число отзывов, телефон.
2. Оба бейджа `2GIS` и `Я.Карты` — одного веса/цвета (один не
   подсвечен жирно, другой не приглушён).

### D2. Только 2GIS

1. Кликни «2GIS» в шапке.
2. Та же компания (она не исчезла — 2GIS-профиль есть).
3. **Бейдж «2GIS» подсвечен** (тонкая брендовая плашка),
   **«Я.Карты» приглушён** (полупрозрачный текст). НЕ зачёркнут.
4. Рейтинг/N отзывов/телефон могут отличаться от того, что было
   на «Все» — потому что сейчас это 2GIS-цифры, а на «Все» был агрегат.
5. **Deeplink** «↗ 2GIS» / «↗ Я.Карты» — должна остаться только
   ссылка «↗ 2GIS».

### D3. Только Я.Карты

1. Кликни «Я.Карты».
2. Бейдж «Я.Карты» подсвечен, «2GIS» приглушён.
3. Рейтинг/отзывы — Yandex-цифры (могут отличаться от D2).
4. Deeplink — только «↗ Я.Карты».

### D4. Возврат на «Все»

Кликни «Все». Карточка возвращается к виду из D1 — оба бейджа ровные,
оба deeplink-а есть.

### D5. Одноисточниковая компания при фильтре

При фильтре «Только 2GIS» найди компанию с **одним** бейджем «2GIS»
(не склеенная). После переключения её вид НЕ должен меняться — нечего
фокусировать.

## E. Drawer карточки — Yandex-отзывы (главное в этом аудите)

Кликни на любую склеенную компанию → справа открывается drawer.

### E1. Блок «Метрики по источникам»

В drawer есть таблица:

| Источник | Рейтинг | Отзывы | Негатив |
|---|---|---|---|
| 2GIS | 5.0 | 5 | — |
| Я.Карты | 5.0 | **50** | … |

🔴 **БАГ если**: у Я.Карт стоит `0 отзывов` для **известной** компании
(см. эталоны ниже). До сегодняшнего фикса там для всех склеенных
компаний было 0.

### E2. Табы отзывов

Под кнопкой «Письмо» есть переключатель табов: `Все (N) / 2GIS (X) / Я.Карты (Y)`.

- Клик на таб «Я.Карты» → должен показать **реальные отзывы из Яндекса**
  (заголовок «Отзывы», список карточек с автором, датой, текстом).
- Если в табе «Я.Карты (Y)» Y > 0, но при клике пишет
  «Отзывов под текущие фильтры не найдено» — 🔴 БАГ.

### E3. Эталонные компании (точно имеют yandex-отзывы)

| Поиск | Компания | Yandex-отзывов в БД |
|---|---|---|
| `стоматология / Балашиха` | Стоматологическая Практика 14*14 | 50 |
| `стоматология / Москва` | Интердентос | 50 |
| `стоматология / Москва` | Зубландия | 51 |
| `стоматология / Москва` | Дантистофф | 51 |

Найди хотя бы 2 из них в соответствующих поисках, открой drawer,
проверь E1+E2.

## F. CSV-экспорт (Excel-friendly)

1. После завершения поиска жми **«Экспорт CSV»** в шапке.
2. Файл скачается с именем `maps_search_NNN.csv`.
3. Открой файл в **Excel** (если Chrome-агент не может запустить
   Excel — открой в **Google Sheets** через `File → Import → Upload`
   с настройкой `Separator: Semicolon`, или просто посмотри как
   браузер отображает первые символы).

**Должно быть:**
- Столбцы РАЗДЕЛЕНЫ: `id`, `name`, `niche`, `city`, `address`,
  `phone`, `website`, `rating`, `reviews_count`, … — не склеены в одну колонку.
- Кириллица **читается**: «Стоматологическая Практика 14*14», «Балашиха» и
  т.п. Никаких `РўРѕ`, `СѓСЃСЃ` — это была старая бага без UTF-8 BOM.

🔴 **БАГ если**: видишь кракозябры либо все колонки в первой ячейке.

## G. Тёмная тема — короткая проверка

1. Включи тёмную тему (луна-иконка в правом верхнем углу).
2. Сегмент `Все · 2GIS · Я.Карты` в шапке: активная кнопка — белая на
   тёмном (или тёмная на светлом, главное — читаемо).
3. На карточке склеенной компании при фильтре `2GIS`: подсвеченный
   бейдж «2GIS» остаётся читаемым, приглушённый «Я.Карты» виден
   (не пропал в тёмное).
4. Drawer-блок «Метрики по источникам» — таблица читается.

## Чего НЕ делать

- **НЕ нажимай «Экспорт CSV» больше 1 раза за поиск** — скачивает файл,
  Chrome-агент может зависнуть на dialog. Если случайно нажал и появился
  диалог «Сохранить как» — отмени.
- **НЕ делай много поисков** — каждый это 1-5 минут парсинга + Yandex
  ходит за SSR-HTML (нагрузка). Достаточно 2 поисков:
  - `стоматология / Балашиха` (для разделов B/C/D/E1)
  - `стоматология / Москва` (для эталонных компаний из E3 — Зубландия/Дантистофф/Интердентос).
- **НЕ паникуй если у компании нет yandex-отзывов** — это норма, не у
  всех они есть. Считай багом ТОЛЬКО для эталонных компаний из E3.
- **НЕ тестируй другие разделы** (ЛК, email-кампании, SEO) — только
  «По картам».
- **НЕ запускай AI-анализ** — у нас известный баг (AI-422), не нужно
  тратить на него попытки.
- **НЕ фикси сам, только описывай в отчёте.**
